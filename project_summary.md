# SME-Ecotech — Short summary (English)

This repository implements **ECOTECH** (*Kiến trúc sư Tài chính*): a React SPA for cash flow, projects, payroll, and approvals, with Express + Vite (dev) and Socket.io.

**Full system context (structure, routing, domain model, data notes)** is documented in Vietnamese here:

- [docs/ECOTECH_CONTEXT_TOAN_HE_THONG.md](docs/ECOTECH_CONTEXT_TOAN_HE_THONG.md)

**Stack (high level):** React 19, Vite 6, TypeScript, Zustand, Tailwind 4, React Router 7, Socket.io, Recharts, `@dnd-kit`, Motion.

**Note:** `@google/genai` is listed in `package.json` but is not imported in application code yet.
